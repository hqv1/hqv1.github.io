    Functional Requirements

    Feature: Create a game

    As a player
I want to create a dominoes game with other friends
so I can enjoy an activity with friends

    Scenario 1: Player starts a game

    Given a player
When she starts a new game
 And she sets some game options
Then a new game is created
 And a unique identifier is returned
 And she can send the identifier to her friends

    Scenario 2: Player joins a game

    Given a player
 And he receives a game identifier
When he uses the game identifier
Then he'll join an existing game

    Scenario 3: All players join the game

    When all players join the game
Then the game starts
 And a round begins

    Feature: Play a round

    As a player
I want to play a dominoes round 
so that I can beat other players

    Scenario 1: Start a round

    Given the game has started
  But a round has not started
When a player starts a round
Then a new round is created
  And tiles are given to each player

    Scenario 2: Player plays her hand

    Given the round has started
  And it is Player's X move
When she makes a move
Then the move is reflected in the round
  And all players can see the move 
  And if she scores points then the points are reflected

    Scenario 3: Round completes

    Given that the player has put down his last dominoes
  And the round is completed
  And the game isn't over yet
When the round is complete
Then another round can begin

    Feature: Complete a game

    As a player 
I want to finish a game
so that I can see who won

    Scenario 1: Game completes

    Given that the round completes
When a player wins the game
Then the player is declare the winner
  And the game is over

    Feature: Game reset

    As a player
I want to store and come back to an existing game
so that I can pick up where I left off

    Scenario 1: Game continues

    Given that a game has started
 And the game hasn't been removed
When a player access the game
Then the game removal timer is reset

    Scenario 2: Game removal

    Given that a game has started
 And the game hasn't been removed
When players have not access the game for a period of time
Then the game is removed

    Feature: Playability

    As a player
I want to play on my computer
 And on my mobile device
So I can play wherever I happen to be
